Steam饰品交易自用脚本与插件

2019-02-11

本文章初始发布日期：2019年2月11日 04:06，请注意内容时效。

在Steam和饰品平台上买卖饰品时，我们经常会发现操作起来不太方便，本文将会整理一些我平时正在使用的脚本与插件。
本期我们为大家带来的脚本是 Steam Economy Enhancer ，它可以帮助你更好地处理你的库存、交易报价，以及Steam社区市场，并且安装使用都十分方便，非常推荐各位安装。

Tips：本文部分内容需要科学上网才能正常使用。

一切的基础——Tampermonkey

Tampermonkey，又称油猴。你可以使用它在你的Chrome等浏览器中运行各类用户脚本，来对网页内容等进行修改，来达到优化页面内容、方便使用等的目的。本文中的大部分脚本也需要依赖这个插件，所以请务必先安装好油猴。

Chrome网上应用商店下载：https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo

Steam Economy Enhancer 简介

Steam Economy Enhancer，可以让你更加方便地管理库存，市场以及交易报价。它能做到的事情很多，例如以下这些：

  • 快速查询库存中各项物品价格

  • 批量快速出售库存中的物品

  • 批量将库存中的物品转换成宝珠

  • 在库存中快速查询市场中那件物品的售卖/求购价格表

  • 交易报价中批量选择自己的物品

  • 交易报价中实时估算双方送出的物品总价值

  • 快捷管理市场挂售中的物品

额外注意事项

由于作者比较的摸，这个脚本的“转换成宝石”功能可能会出现无法使用的情况，这个时候则需要对这个脚本的源代码进行修改，以解决这个问题。

首先，点开油猴图标，点击“管理面板”。

接着点击“Steam Economy Enhancer”来进入代码编辑界面。

使用“Ctrl+F”搜索“Due to”，并定位到图上一段代码：

if (err != ERROR_SUCCESS) {
    logConsole('Failed to get gems value for ' + itemName);
    logDOM(padLeft + ' - ' + itemName + ' not turned into gems due to missing gems value.');

    $('#' + item.appid + '_' + item.contextid + '_' + itemId).css('background', COLOR_ERROR);
    return callback(false);

item.goo_value_expected = parseInt(goo.goo_value);</code></pre>

将这两段代码注释掉，并新增这一行代码：

<pre class="wp-block-code"><code>item.goo_value_expected = 100;</code></pre>

最终效果如下：
