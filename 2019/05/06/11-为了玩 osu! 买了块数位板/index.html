    为了玩 osu! 买了块数位板

    本文章初始发布日期：2019年5月6日 13:00。

    15年的时候听说了 osu! 这款游戏，注册了个账户，进去玩了一圈感觉好难就把游戏删了，直到最近的某一天突然又想玩了，于是决定回坑。
    从注册到现在，我的游戏时长一共才十小时左右，加上比较手残，平时基本只打打三星图的样子，非常的菜。不过月初手上正好有点闲钱，就决定买一块便宜的数位板来试试看。

    开箱

    在工作室的第三天，我收到了这块全新的 Wacom CTL-472 数位板，淘宝花了我200出头，快递还是顺丰的，隔日就送到了。

    数位板使用了一个很小的纸盒包装，很简约，非常对我胃口。

    Awesome！这是我独享的moment！来，给你们也看看！

    盒子里还有一根数据线，一支笔，和说明书质保卡之类的几个东西。

    再来康一下板子本体

    最后康一下送的赠品

    • 延长线以及笔盒

    • 收纳袋

    使用前准备

    首先试了一下官方驱动，发现有点小问题，例如一些延迟之类的，于是去咨询了一下群内的osu大佬，得知在Github上有一个打osu用的专用驱动，试了一下确实改进很大，那我就顺便介绍下这个驱动吧。

    1. 卸载掉官方驱动，并重启一下电脑。
    2. 进入这个页面下载最新版本的驱动，并将全部文件解压出来。
    3. 打开 install_vmulti_driver.bat 并安装前置驱动
    4. 启动 TabletDriverGUI.exe ，并设置好映射区域，接着点击 Save settings 来保存设置，大功告成！
    5. 最小化窗口，就可以开始打osu了，不过这个软件默认不开机自启，下次打osu之前记得要再把它打开鸭！

    开玩！

    接着就可以开始了~

    试了一下用板子打了几首，没了鼠标的重量感觉手抖的厉害，但是因为笔比较轻，拉枪比之前用鼠标也快了很多，并且转盘装起来大爽！用鼠标只能打三星出头点的图，换了板子之后四星也能摸一摸了，非常舒适。

    我录了一小段用板子打的视频，不过别忘了！我是只玩了十小时的萌新！千万别喷我菜！我会哭给你看的！！
    （视频在油管，内地是要挂梯子才能康到下面的视频der）

    最后没啥特别要说的，就祝大家身体健康吧！
